# WakaTime Desktop Application


Note: this project is not finished and not yet working.


A universal installer for all WakaTime [plugins][plugins].

## Installation

    git clone https://github.com/wakatime/desktop.git
    npm install -g yarn
    yarn install
    yarn run build
    yarn start
